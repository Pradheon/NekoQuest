Basic Help

To advance through the game, press the enter key. When at a menu, use the arrow keys to select a choice and enter to activate it.

Main Menu

When at the Main Menu, navigate the menu options with the arrow keys and press enter key to select a choice. The game menu gives the following choices:

Backspace
Returns to the last screen.
Preferences
Changes the game preferences (options/configuration):
Display
Switches between fullscreen and windowed mode.
Music and Sound Volume
Controls the volume of the Music and Sound effect channels, respectively. The further to the right these are, the louder the volume.
Controls
Shows all the controls for the game.
Main Menu
Returns to the main menu, ending the current game.
Quit
Exits the game; the game will be closed and ended.

Key and Mouse Bindings

Enter
Advances through the game, activates menu choices, buttons, and sliders.
Space
Shots neko pellets.
Arrow Keys
Navigates between menu choices and character movement.
Up Arrow Key
Causes player to jump.
Left Arrow Key
Causes player to move left.
Right Arrow Key
Causes player to move right.
Alt-M, Command-H
Hides (iconifies) the window.
Alt-F4, Command-Q
Quits the game.
